About Me

I'm a DevOps engineer in India who builds cloud-native infrastructure with security wired in from the first commit — not bolted on at the end.

My CI/CD pipelines gate every container build on Dockle, Checkov, and Semgrep, so a vulnerability or misconfiguration stops the build before it reaches production. I build images with Kaniko, orchestrate the flow through GitHub Actions, and define AWS infrastructure as reusable Terraform modules — so environments stay versioned and reproducible instead of clicked together by hand.

My approach is simple: automate the repeatable parts, make security the default rather than a review-stage afterthought, and keep everything reproducible from code. Right now I'm building out an advanced security-first CI/CD setup and learning Kubernetes the Kustomize way.
